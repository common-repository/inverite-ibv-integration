== Installation ==

To use the plugin, install it from the Wordpress plugin directory or manually following these steps:

1. Install or Upload the plugin to your Wordpress installation.
1. Activate the plugin through the 'Plugins' screen in WordPress.
1. Use the shortcode like so: `[inverite-iframe site-id="<site-id>"]`.
1. Enjoy!
